**核心功能**

**抓取能力**
- 输入关键词自动打开浏览器抓取小红书搜索结果
- 支持逗号分隔多个关键词，一次批量抓取
- 可调节滚动次数控制抓取深度，滚得越多图越多
- 同时抓取标题、点赞数、评论数、原帖链接

**筛选 & 排序**
- 抓取前可设置最低点赞数，直接过滤低热度内容
- 抓取后还能二次筛选、按点赞数排序，只看爆款

**图片预览**
- 缩略图异步加载，界面不卡顿
- 鼠标悬停放大预览（500×500）
- 右键可打开全屏大图（自适应屏幕85%）

**下载管理**
- 全选 / 取消全选，勾选批量下载
- 下载记录持久化，已下载过的图自动标记提醒
- 图片按日期自动归档到子文件夹
- 保存路径自由选择，且记忆上次位置

**其他细节**
- 登录状态持久保存，不用每次重新登录
- 右键菜单可复制图片链接、帖子链接，或直接打开原帖
- 一键打开保存文件夹，随时查看已下载内容

---

**技术亮点**（如果对方懂技术可以提）
- 基于 Playwright 驱动真实浏览器，绕过普通爬虫的反爬限制
- 图片加载与 UI 完全分离，多线程异步，界面始终流畅
- 去重基于 URL 主路径的 MD5 哈希，token 变化也能准确识别同一张图
